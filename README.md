## ğŸ“Œ Project Summary

**SOKOFARM-GAME** is a Sokoban-style puzzle game designed for beginners. The goal is to place seeds in the correct spots to make them grow. The game contains 25 puzzles, playable manually or automatically using AI search algorithms with optional heuristic strategies.

The system supports:
- Manual gameplay with keyboard controls  
- Automatic puzzle solving using BFS, DFS, UCS, A*, Hill Climbing, and heuristic search  
- Step-by-step visualization of algorithmic solutions  
- Educational gameplay combining puzzles and AI
